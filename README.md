GameBall
========
GameBall - two player socker written in C++ using SFML

#Features
- Changable physics
- Changable avatars
- Time or score based winning
- Kicking

#Controls
###Player 1.
- Up/Down/Left/Right - W/S/A/D
- Kick - Q

###Player 2.
- Up/Down/Left/Right - Arrows
- Kick - Space
